Pairwise Coupling and Pairwise with Majority Voting

 

The schemes require a binary classifier for each possible pair of classes. The decision function of the SVM classifier for -to- and -to- has reflectional symmetry in the zero plane. Hence only one of these pairs of classifier is needed. The total number of classifiers for a K-class problem will then be K(K-1)/2. The training data for each classifier is a subset of the available training data, and it will only contain the data for the two involved classes. The data will be reliable accordingly, i.e. one will be labelled as +1 while the other as -1.

These classifiers will now be combined with some voting scheme to give the final classification results.   • Estimates the pairwise probability directly
    This is much simpler compared to the previous method. If only the pairwise probabilities are required, then they can be estimated directly without obtaining a proper p.d.f. estimate. A pseudo p.d.f. can be use to estimate the pairwise probabilities. The choice of the pseudo p.d.f. is based on the following factors:

    • The classification boundary of a SVM classifier is at f(x)=0 and the classification confidence increases with the magnitude of the decision function.
    • The pairwise probability is a sigmoid function with value from 0 to 1 and the decision boundary is the point where the value of the sigmoid function is 0.5.

    • It represents the decision boundary of the SVM accurately, since the decision boundary is always at f(x)=0.
